# GDJ105

## Program execution

to execute the the compiled jar, please execute the command bellow :

```bash
$> mvn exec:java
```

or :

```bash
$> java -jar gdj105-0.0.1-SNAPSHOT.jar
```

After full build, on the windows platform, you can execute the `target/ GDJ105.exe` executable file.

```bash
c:\> target\GDJ105.exe
```
