### Goal

As we 've got now some great piece of game code, we can add some new objects as graphical objects to create some new State like TitleState and MenuState. Let's create TextObject, BackgroundObject, ButtonObject and so on...

## Compile

To compile the full project, please execute the followin' command :

    $> mvn clean install


## Execute

to execute the the compiled jar, please execute the command bellow :

    $> mvn exec:java

or :

    $> java -jar gdj105-0.0.1-SNAPSHOT.jar
